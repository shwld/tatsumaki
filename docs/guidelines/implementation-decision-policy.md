# Implementation Decision Policy（実装判断）

リポジトリ横断で、実装方針のトレードオフを揃える。`CLAUDE.md` と `AGENTS.md` は本文を複製せずここへのポインタとする。

## 長期最適優先

- デフォルトは短期最適ではなく**長期最適**を優先する。
- 実装提案時は短期案と長期案がある場合でも**長期案を第一提案**とする。

## 短期実装を許容する場合

次のときのみ短期実装を選ぶことができる。

- ユーザーが明示的に短期実装を要求したとき
- 本番障害などの緊急対応で一時的な止血が必要なとき

この場合でも、あわせて**長期移行パス**（次の差分での正攻法・計画への言及）を明示する。
